The Living Pages V2.7 Daily Legacy Studio

Purpose:
A private offline-first legacy journal that encourages daily writing without forcing a finished poem. The user can leave one true thing: a free page, guided memory path, daily reflection, photograph story, voice note, unfinished fragment, working poem, or legacy letter.

V2.7 improvements:
- Rebuilt the whole app experience around "What do you want to leave today?"
- Added clearer daily choices: Write Freely, Write With Guidance, Today's Page, Photo or Voice, Continue Later.
- Improved guided writing UX: type any memory anchor, choose a door, then choose one of three writing paths.
- Prompts do not enter the draft automatically. The user must press "Use This Path."
- Reworked guidance to feel like writing paths instead of one confusing prompt.
- Kept V2.4 clear Save / Save as New / Replace / Delete behavior.
- Kept photo and voice Save / Replace / Delete actions.
- Kept offline-first PWA behavior and Firebase-ready files.
- True Mentor is still untouched/placeholder until a secure AI backend is built.

Installation:
Upload all files in this folder to GitHub Pages. Open the live app once online so the service worker caches it. Then test by turning internet off and reopening the app.

Important:
Text saves first on device. Photos and voice can be stored locally with the page and synced later when Firebase is enabled and the user is logged in. Keep Export Backup available for legacy safety.
