THE LIVING PAGES V2.2 — TRUE MENTOR LEGACY JOURNAL

Purpose:
A private legacy journal for daily writing, poems, memories, photographs, and voice notes. The app protects authorship: AI guidance may mentor, question, reflect, and clarify, but it must not write the poems for Aileen.

What's new in V2.2:
- Write Freely, Guide Me Gently, Today's Living Page, Preserve a Photograph, Record My Voice.
- Structured but open prompts: use categories or type any memory anchor such as sand, rock, gate, dress, room, vacation, sound, or object.
- Help Me Continue, Help Me Finish, Make it Clearer, Make it More Like Me, Ask Me Deeper Questions.
- Store for Later as Fragment, Memory Seed, Unfinished Page, Working Poem, Legacy Poem, Letter Note.
- Offline PWA structure: manifest.json + service-worker.js.
- Firebase-ready config, Firestore/Storage rules, and AI mentor endpoint placeholder.

Important:
For the true AI mentor, use a secure Cloud Function endpoint. Do NOT put OpenAI/API secret keys inside GitHub or browser JavaScript.

Offline testing:
Upload the full folder to GitHub Pages. Open the live link once online. Then turn internet off and reopen. Service workers do not work reliably from raw file:// opens.
