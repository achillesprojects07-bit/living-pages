The Living Pages V1.1 QUICK POEM FLOW

Purpose:
A private legacy studio for recovering Aileen's poetic voice and preserving poems, fragments, photos, and voice notes for her children.

What changed in V1.1:
- Added Create Something Today / 10-Minute Poem Flow.
- Added 5-prompt guided start: memory, photo, or feeling.
- Added simple Editing Window.
- Added three voice-protecting guidance buttons: Make it clearer, Make it more like me, Help me finish.
- Added save levels: Fragment, Memory Seed, Working Poem, Legacy Poem, Letter Note.
- Added optional photo attachment and browser voice recorder.
- Added Voice Doors and Voice Map.
- Still local-only: saves in browser localStorage on the current device.
- Firebase cloud vault is not yet connected.

Important rule:
AI does not write the poem. The app guides, mirrors, asks questions, and helps edit gently while preserving authorship.

How to use:
1. Open index.html in a browser.
2. Click Start the 10-Minute Poem Flow.
3. Answer the five prompts.
4. Arrange your own lines in the Editing Window.
5. Use the guidance buttons only if needed.
6. Save to archive.
7. Use Export Legacy Backup often.

Notes:
- Voice recording requires microphone permission in the browser.
- Photos and audio in this version are saved locally in the browser and can make the browser backup larger.
- Export backup before clearing browser data or moving devices.
